**Raycaster_C++**

Простой рейкастер, написанный на C++, имитирующий 3D-графику с помощью техники raycasting, аналогичной использованной в оригинальной игре Wolfenstein 3D.

Особенности:

- Рендеринг 3D-подобного вида из 2D карты
- Простое освещение и затенение
- Миникарта для навигации
- Плавное движение и вращение камеры

Разработано как образовательный проект для изучения компьютерной графики и C++.
